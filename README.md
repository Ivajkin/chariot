# Chariot

Chariot is build optimizer.

## Features

* JS Linting
* JS Minification
* RequireJS Optimization

## Usage

1. Install NodeJS and NPM
2. On the command line execute `npm install -g chariot`
3. Navigate into your web project and place a weyland-config file at the root.
4. From your project directory execute `chariot build`
